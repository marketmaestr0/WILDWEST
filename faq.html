Token Launches

How much does it cost to launch a token?

Token creation is free - you only pay gas fees and a few cents to start the liquidity pool.

How does the dev buy period work?

During the first 5 minutes after token launch, the developer has an exclusive window to buy the token and no one else can trade until that 5 minute window is over. Once the 5 minute window is over, the public is free to trade.

What information do I need to launch a token?

You only need a token name and symbol to launch.

Which networks can I launch tokens on?

Currently only Base Network is supported. We are working on adding Solana and then more EVM chains.

What happens after I launch my token?

The source code is automatically sent to BaseScan to be verified and a liquidity pool is created on the corresponding DEX (Uniswap, etc.).

Can I edit my token information after launch?

Yes! Once your token is created, go to the main page and connect your wallet to edit your project card. Your logo, banner, and links can all be added and edited at any time.

Platform Utility & Features

What is token locking and how does it work?

Anyone can lock any token and choose any beneficiary - themselves or another wallet. You can lock for any amount of time paying only the gas fees to create the contract. This is a free service.

What is the Developer Tracker feature?

This feature is currently under development and will be used to track developer wallet holdings.

How does the trending system work?

Our trending algorithm evaluates tokens based on multiple factors:

  • Trading Volume: 24-hour trading activity (30% weight)
  • Transaction Count: Number of individual trades (30% weight)
  • Market Capitalization: Total token valuation (30% weight)
  • Recency Factor: Newer tokens receive slight boost (10% weight)

What are the trading fees?

There is a 1% fee on the liquidity pool. The project keeps all of the token rewards from this pool and splits the ETH rewards with the platform - equating to around 75% for the project depending on buy and sell pressure.

How do I connect my wallet?

On mobile, connect your wallet inside of your wallet app using the integrated wallet browser. On desktop, you should have wallet extensions installed inside of your browser.

Is Wild West Launchpad secure?

Yes, the factory contracts are verified and opensource for full transparency.
